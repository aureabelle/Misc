Messages

Messages should not be confused with regular contents. User-invoked or messages that came from the system are temporary and are meant to appear as a consequence of an action. It is recommended to use these messages sparringly and in context because they will lose their effectiveness if overused. Consistent use of these messages can also be effective to get users familiar with the behavior of the UI.

Global Messages (User-invoked messages)

Global messages are used for user-invoked messages, such as when a user submitted a form and in return gets a success message. These type of messages are designed to accommodate single-phrase messages. They can be paired with any icons or use them as they are. To use any of these messages, simply attach class="message" plus msg-success or msg-fail or msg-warning or msg-reminder to a block element.

Do not use global message type for messages that consist of multiple blocks elements.

Inline Field Buttons

Inline "field buttons" are buttons that can be used when retrieving data from the server prior to the actual form submission.

This type of button can only be either a button type="button" or input type="button" with class="btn". It cannot be a button type="submit" or an input type="submit" because it will inherit the default styling for buttons of type=submit, which should only be for the actual submit actions of a form. This is so as to keep consistency of the usage throughout.

<button type="button" class="btn">Find</button>
---
<input type="button" value="Button" class="btn" />

Default Form Action Buttons

Default form action buttons are the main buttons that can normally be found in a regular form. Use either button or input with type="submit" or type="button" or type=reset, no class attribute attached. Should there be a case when an action button be disabled, include the attribute disabled="disabled" so that the button behaves as it should and inherits the style for the state.

Please not that type=submit button should always be different in style (green) from the other buttons (grey) for easy recognition.
